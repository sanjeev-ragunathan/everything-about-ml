# Agentic AI (2024–2025)

## The Story

AI stops just answering and starts doing. Tool use, function calling, multi-step planning. Agents browse the web, write code, manage tasks. The shift from copilot to autonomous worker begins.

## Key Concepts

- **Tool use** — Models that call functions and APIs
- **Function calling** — Structured output that triggers actions
- **Planning** — Break goals into sub-tasks and execute
- **RAG** — Retrieval-Augmented Generation, grounding answers in real data

## What You Build

- AI agent that can call tools, search, execute code
- Chain actions together for multi-step problem solving

## Run

```bash
cd ai-agent
python ai_agent.py
```

## Key Insight

A model that can act on the world is fundamentally different from one that can only describe it. Agency changes everything.
