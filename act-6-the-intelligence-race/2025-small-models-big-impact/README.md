# Small Models, Big Impact (2025)

## The Story

Efficient models rival giants. DeepSeek shows open-source can match closed labs. Quantization, distillation, and pruning let models run on phones and edge devices. The democratization of AI accelerates.

## Key Concepts

- **Quantization** — Reduce precision (32-bit → 4-bit) to shrink models
- **Knowledge distillation** — Train a small model to mimic a large one
- **Pruning** — Remove unnecessary weights
- **Edge deployment** — Run models on devices, not servers

## What You Build

- Take a large model and make it tiny
- Quantize, distill, and benchmark speed vs accuracy

## Run

```bash
cd model-optimization
python model_optimization.py
```

## Key Insight

The best model isn't always the biggest. The best model is the one that solves your problem within your constraints.
