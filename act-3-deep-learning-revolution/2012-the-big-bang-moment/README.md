# The Big Bang Moment (2012)

## The Story

ImageNet: 1000 classes, 1.2 million images. AlexNet wins by a 10-point margin. GPUs, ReLU, dropout. The second AI winter ends permanently. Deep learning is real.

## Key Concepts

- **GPU training** — Parallelism changes everything
- **ReLU** — Simple activation, massive impact
- **Dropout** — Regularization by randomly silencing neurons
- **Data augmentation** — More data from the data you have

## What You Build

- AlexNet-style architecture on CIFAR-10
- Train on GPU and watch accuracy jump vs LeNet

## Run

```bash
cd alexnet
python alexnet.py
```

## Key Insight

Scale was the missing ingredient. The algorithms existed for decades — it was compute and data that unlocked them.
