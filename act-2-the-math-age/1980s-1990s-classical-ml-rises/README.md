# Classical ML Rises (1980s–1990s)

## The Story

The field learned you don't need neural nets for everything. SVMs, decision trees, k-NN, logistic regression — elegant math, powerful results. Statistics and ML merged. The Kaggle era of feature engineering began.

## Key Concepts

- **SVM** — Find the widest margin between classes
- **Decision trees** — Split data by asking questions
- **Ensembles** — Combine weak learners into strong ones
- **Regularization** — Prevent overfitting
- **Feature engineering** — The art of choosing what to feed the model

## Key Insight

Classical ML dominates when you have structured data, small datasets, or need interpretability. Neural nets aren't always the answer.
