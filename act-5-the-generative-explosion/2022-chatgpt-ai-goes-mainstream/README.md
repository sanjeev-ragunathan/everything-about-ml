# ChatGPT — AI Goes Mainstream (2022)

## The Story

GPT-3.5 wrapped in a chat interface. 100 million users in 2 months — the fastest-growing app in history. The secret ingredient wasn't the model — it was RLHF, Reinforcement Learning from Human Feedback, which made the model helpful, harmless, and honest.

## Key Concepts

- **RLHF** — Train a reward model on human preferences, then optimize against it
- **Instruction tuning** — Teach models to follow directions
- **Alignment** — Making AI do what humans actually want
- **Reward modeling** — Scoring outputs by human preference

## What You Build

- Simplified reward model
- See how human preferences shape AI behavior

## Run

```bash
cd rlhf
python rlhf.py
```

## Key Insight

Raw capability isn't enough. A powerful model that ignores your intent is useless. Alignment turns a language model into an assistant.
